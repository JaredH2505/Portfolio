Projects

Below are some of the projects that I have worked on

Django based website

The purpose of this site is to be student assignment submission and grading site. It supports user rolls, authentication, and server-based rendering.

TCP Emulator

This project was a recreation of the basics of TCP networking build in a simulator provided by the course professor. My code needed to be able to handle packet loss, packet corruption, and variable transmission speed.

Smart Watch Interface

This project was an assignment to design an input method for text on a smart watch. This designed was created as a prototype in processing and then tested against other prototypes for usability and speed.

Parallel Computing

This project is a collection of assignment from my time in the CS 4230 parallel computing class. It was my introduction to parallel computing with GPUs via CUDA and networked computer via MPI-io.

Computer Security

This is a final project for CS 4440 covering computer security. This is my code for various attacks against the included websites which include SQL injection, cross site scripting, and hash poising. These different attacks had to be engineered to overcome various safe guards.

EcoSim

EcoSim was my capstone project for my time at University of Utah. It is an economic simulation system based on trend lines, fully integrated into the unity game engine with a demo game to show off what how it can augment gaming experiences.
